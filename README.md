# afterschool-registration-website

##Overview##  
A web application designed to help guardians easily register their children for the afterschool program and view activites. It provides a simple, user-friendly interface for viewing available clubs, submitting registration forms, and managing student participation. The goal of this website is to make registration more efficient, organized, and accessible for both families and program administrators.
